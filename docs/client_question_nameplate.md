# ネームプレート価格設定についてのご確認

ネームプレートの価格設定について、1点ご確認させてください。

現在のシステムでは、ネームプレートの価格を「デザインあり」「デザインなし」の2パターンで切り替える仕組みになっています。

ご指摘いただいた「再版でデザインなし（増刷のみ）」の場合の計算式を拝見したところ、以下のように「版代」が別途発生することが分かりました。

■ 再版・デザインなし（1名）の場合
デザイン費：0円
版代：1,700円
制作費：1,000円 × 1名
送料：1,000円
合計：3,700円

この「版代」は現在のシステムの価格テーブルに項目がなく、対応するためにはシステムの改修が必要です。

つきましては、以下の点をご確認いただけますでしょうか。

---

**ご確認事項**

1. 「版代」は名数（1名・2名・3名…）に関わらず一律1,700円でしょうか。それとも名数によって変わりますか？

2. 「再版・デザインなし」の場合、制作費（1,000円/名）は名数に比例して増えていく理解でよいでしょうか。

3. 「版代」が発生するのは「再版・デザインなし」のケースのみでしょうか。それとも他のケースでも発生しますか？

4. 現在の価格テーブルに「版代」を独立した項目として追加する形でよいでしょうか。それとも、再版専用の別メニューとして切り分けた方がよいでしょうか。

---

上記をご確認いただいたうえで、システムへの反映を進めます。
