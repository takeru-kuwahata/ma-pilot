## 移行手順

### 1. GitHub リポジトリ転送

1. GitHubの `takeru-kuwahata/ma-pilot` を開く
2. Settings → General → Danger Zone → **Transfer repository**
3. 移行先のGitHubアカウント名を入力して転送
4. VercelのGitHub連携が自動で新リポジトリを追う（要確認）

### 2. Supabase 組織転送

1. Supabaseダッシュボード（https://supabase.com/dashboard/project/ecdzttcnpykmqikdlkai）を開く
2. Settings → General → **Transfer project**
3. 移行先のSupabaseアカウントのメールアドレスを入力して転送
4. **⚠️ データはそのまま引き継がれる**

### 3. Vercel プロジェクト転送

1. Vercelダッシュボードで `ma-pilot` プロジェクトを開く
2. Settings → General → **Transfer Project**
3. 移行先のVercelアカウントに転送
4. GitHubリポジトリの再連携が必要な場合あり
5. 環境変数は自動で引き継がれる

### 4. Render サービス転送

1. Renderダッシュボード（https://dashboard.render.com）を開く
2. `ma-pilot` サービスを開く
3. Settings → **Transfer Service**
4. 移行先のRenderアカウントのメールアドレスを入力
5. 環境変数は引き継がれる

| 変数名 | 内容 |
|---|---|
| SUPABASE_URL | SupabaseプロジェクトURL |
| SUPABASE_KEY | Supabase Service Roleキー |
| RESEND_API_KEY | Resend APIキー（MA名義） |
| RESEND_FROM_EMAIL | dr@medical-advance.com |
| GOOGLE_MAPS_API_KEY | Google Maps APIキー |
| VITE_BACKEND_URL | https://ma-pilot.onrender.com |

## Renderの有料プランについて

**無料プランの制限：** 15分間アクセスがないと自動スリープ（起動に最大50秒）

**推奨プラン：** Starter（$7/月）→ スリープなし・常時起動

移行前に医療Design Workshopで有料プランに切り替え済み。
移行後はメディカルアドバンスのアカウントで同プランを継続してください。
