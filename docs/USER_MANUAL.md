## ログイン・アカウント作成

### 初回ログイン

1. `https://ma-pilot.vercel.app`にアクセス
2. メールアドレス・パスワードを入力
3. 「ログイン」ボタンをクリック

### アカウント作成（招待経由）

1. 医院オーナーから招待メールを受信
2. メール内のリンクをクリック
3. パスワードを設定
4. 「アカウント作成」ボタンをクリック

### パスワードリセット

1. ログイン画面で「パスワードを忘れた方」をクリック
2. メールアドレスを入力
3. 送信されたリンクからパスワードを再設定

## 経営ダッシュボード

### 概要

月次経営データをグラフとKPIで可視化します。

### 主要KPI

- **総売上**: 当月の総売上金額
- **売上成長率**: 前月比の成長率（%）
- **総患者数**: 当月の総患者数
- **患者単価**: 総売上 ÷ 総患者数
- **営業利益**: 総売上 - 総コスト
- **利益率**: 営業利益 ÷ 総売上 × 100

### グラフ

- **売上推移**: 過去12ヶ月の売上トレンド
- **患者数推移**: 新患・再診患者数の推移
- **コスト構成**: 人件費・材料費・固定費の割合

### アラート

- **新患数減少**: 前月比10%以上減少で警告
- **利益率低下**: 利益率20%未満で警告
- **患者単価低下**: 患者単価が前月比5%以上減少で警告

## 基礎データ管理

### 月次データ入力

1. 左メニュー→「基礎データ管理」
2. 「新規データ入力」ボタンをクリック
3. 年月を選択（YYYY-MM形式）
4. 以下の項目を入力:
   - 総売上
   - 保険診療売上
   - 自費診療売上
   - 新患数
   - 再診患者数
   - 人件費
   - 材料費
   - 固定費
   - その他経費
5. 「保存」ボタンをクリック

### CSV一括取込

1. PILOTから月次データをCSVエクスポート
2. 「CSV取込」ボタンをクリック
3. ファイルを選択してアップロード
4. 取込結果を確認（成功件数・エラー件数）

#### CSVフォーマット

```csv
year_month,total_revenue,insurance_revenue,self_pay_revenue,new_patients,returning_patients,personnel_cost,material_cost,fixed_cost,other_cost
2024-01,5000000,3000000,2000000,50,200,2000000,800000,500000,300000
```

## 診療圏分析

### 分析実行

1. 左メニュー→「診療圏分析」
2. 分析半径を選択（1km / 2km / 3km）
3. 「分析実行」ボタンをクリック
4. 結果表示（約30秒）

### 分析結果

- **人口統計**: 年齢別人口分布
- **競合医院**: 診療圏内の歯科医院リスト
- **潜在患者数推定**: 診療圏内の潜在患者数
- **市場シェア**: 競合医院との相対的なシェア

### 地図表示

- 自医院（青いピン）
- 競合医院（赤いピン）
- 診療圏（円）

## 経営シミュレーション

### シミュレーション実行

1. 左メニュー→「経営シミュレーション」
2. 目標値を入力:
   - 目標売上（円）
   - 目標利益（円）
   - 達成期間（ヶ月）
3. 現状パラメータを入力:
   - 現在のチェア数
   - 現在のスタッフ数
   - 患者単価
4. 「シミュレーション実行」ボタンをクリック

### 結果表示

- **必要患者数**: 目標達成に必要な月間患者数
- **必要設備**: チェア増設数
- **必要人員**: スタッフ増員数
- **戦略提案**: 具体的なアクションアイテム

### 戦略提案例

- 「チェア2台増設により月間患者数+30%」
- 「歯科衛生士2名採用で自費診療比率向上」
- 「マーケティング強化で新患獲得+20名/月」

## レポート生成・管理

### レポート生成

1. 左メニュー→「レポート生成・管理」
2. レポート種類を選択:
   - 月次レポート
   - 四半期レポート
   - 年次レポート
3. 対象期間を選択
4. フォーマットを選択（PDF / CSV）
5. 「生成」ボタンをクリック
6. ダウンロードリンクをクリック

### レポート内容

#### 月次レポート（PDF）

- KPIサマリー
- 売上・患者数グラフ
- 前月比較
- アラート一覧

#### 年次レポート（PDF）

- 年間KPI推移
- 四半期比較
- 診療圏分析結果
- シミュレーション結果

### レポート履歴

- 生成日時、種類、ダウンロードリンクを一覧表示
- 過去3ヶ月分を保持

## 医院設定・スタッフ管理

### 医院基本情報編集

1. 左メニュー→「医院設定」
2. 以下の項目を編集:
   - 医院名
   - 郵便番号
   - 住所
   - 電話番号
   - 開業日
   - チェア数
   - スタッフ数
3. 「保存」ボタンをクリック

### スタッフ招待

1. 左メニュー→「スタッフ管理」
2. 「スタッフ招待」ボタンをクリック
3. メールアドレスを入力
4. 権限を選択:
   - 医院編集者（データ入力・編集可）
   - 医院閲覧者（閲覧のみ）
5. 「招待メール送信」ボタンをクリック

### 権限変更

1. スタッフ一覧から対象を選択
2. 「権限変更」ボタンをクリック
3. 新しい権限を選択
4. 「保存」ボタンをクリック

### スタッフ削除

1. スタッフ一覧から対象を選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「はい」を選択

## 印刷物注文

### 注文方法（相談フォーム）

1. 左メニュー→「印刷物注文」
2. 「相談フォーム」を選択
3. 必須項目を入力:
   - クリニック名
   - メールアドレス
4. 任意項目を入力:
   - 商品種類（診察券、名刺等）
   - 数量
   - 納期希望
   - 備考
5. 「送信」ボタンをクリック
6. 受付完了メールを受信

### 再注文（見積もり自動計算）

1. 左メニュー→「印刷物注文」
2. 「再注文」を選択
3. 全項目を入力:
   - クリニック名
   - メールアドレス
   - 商品種類
   - 数量
   - 仕様（角丸半径、加工種類等）
   - 納期希望
4. 「見積もり計算」ボタンをクリック
5. 見積もり金額を確認
6. 「承認」ボタンをクリック
7. 決済方法を選択（クレジットカード / 請求書払い）
8. 注文完了

### 注文履歴確認

1. 左メニュー→「印刷物注文履歴」
2. 過去の注文を一覧表示
3. 詳細ボタンで注文内容確認
