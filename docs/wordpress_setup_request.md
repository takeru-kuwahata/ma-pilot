# WordPressの設定変更のお願い

## お願いしたい内容

MA-Pilotシステムとの連携のため、WordPressの設定を2点変更していただく必要があります。

---

## ①「MA-Pilot連携用」アカウントの権限変更

新しく作成した「MA-Pilot連携用」アカウント（`info@medical-advance.com`）のWordPress内での権限を、**「管理者」**に変更してください。

**変更手順：**
1. WordPress管理画面にログイン
2. 左メニュー「ユーザー」→「ユーザー一覧」
3. `info@medical-advance.com` のアカウントを選択
4. 「権限グループ」を **「管理者」** に変更
5. 「ユーザーを更新」ボタンをクリック

**理由：**
現在の権限では、システムが自動でWordPressアカウントを作成する権限が不足しています。管理者権限が必要です。

---

## ②Xserver側のアクセス制限の確認

システムのサーバー（Render.com）からWordPressへのアクセスが、Xserver側のセキュリティ設定でブロックされている可能性があります。

以下をXserverの管理パネルで確認・対応をお願いします。

**確認手順：**
1. Xserverのサーバーパネルにログイン
2. 「アクセス制限」または「WAF設定」を確認
3. 外部からのREST APIアクセスが許可されているかを確認

**もし不明な場合は、**Xserverのサポートに「WordPress REST APIへの外部サーバーからのPOSTアクセスを許可したい」とお問い合わせいただくのが確実です。

---

## 対応後のご連絡

上記2点の対応が完了しましたら、ご連絡ください。完了後に動作確認を行います。
