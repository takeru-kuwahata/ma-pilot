# AI開発向け助成金・補助金 調査レポート

**作成日：2026年6月1日**
**作成：日本医療デザインセンター 桑畑**
**宛先：メディカルアドバンス 本多社長**

---

## 背景

弁理士の方から「AIの開発で最大2000万円・補助率100%の助成金がある」という話があったとのことで、自社で全方位リサーチを行いました。

---

## 結論（先に）

**弁理士の方の情報はおおむね実在の制度を指している可能性がありますが、条件に合う制度は一部限定的です。弁理士を通さなくても、既存のパートナー体制で申請できます。**

---

## 「最大2000万円・100%補助」の正体

「補助率100%・上限2000万円」を完全に満たす制度として確認できたのは以下の1つです。

### SBIR推進プログラム フェーズ1（JST実施）

| 項目 | 内容 |
|---|---|
| 補助率 | **100%** |
| 上限額 | **2000万円** |
| 実施機関 | 科学技術振興機構（JST） |
| 対象者 | **設立15年以内**のスタートアップ・研究開発型中小企業 |
| 公式URL | https://www.jst.go.jp/start/sbir/index.html |

**→ メディカルアドバンスは創業24年目のため対象外。**

弁理士の方がこの制度を念頭に話していた可能性がありますが、御社には適用できません。

## 御社が実際に使える制度（現実的な候補）

### 1. ものづくり補助金 ★本命

| 項目 | 内容 |
|---|---|
| 補助上限 | **最大3,500万円** |
| 補助率 | 中小企業：1/2 ／ 小規模事業者：2/3 |
| 対象 | 革新的な製品・サービス開発（AI・DXによる新規事業含む） |
| 次回締め切り | **2026年8月以降**（現在公募準備中） |
| 採択率 | 30〜40%台 |
| 公式URL | https://portal.monodukuri-hojyokin.jp/ |

**御社との相性：最良**

MA-Pilotの新機能（AIエージェント・ベンチマーク比較等）や、ma-aiプロジェクト（医師データ管理AIシステム）の発展版を「革新的サービス開発」として申請できます。弁理士費用・ソフトウェア開発費・サーバー費用も補助対象経費に含まれます。

**勝ち目：高**（社労士・中小企業診断士パートナーによる申請書の質が直接採択率に影響します）

- 上限450万円と規模が小さい

## 弁理士について

弁理士は本来「特許・知財」の専門家です。補助金申請書類の作成は**行政書士の独占業務**であり、弁理士が申請代行を行うと行政書士法違反になる可能性があります。

**今回の対応方針：**
- 弁理士の方へは「制度の正式名称と公募要領のURLを確認する」程度にとどめる
- 申請実務は既存パートナー（社労士・中小企業診断士）で完結させる
- 弁理士とのビジネス関係を深める必要はなし
