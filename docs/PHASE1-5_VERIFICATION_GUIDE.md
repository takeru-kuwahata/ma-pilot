## ステップ1: クリニックを登録する（管理者として）

### やること
5つのクリニックを登録します。

### 手順

1. https://ma-pilot.vercel.app にアクセス
2. メールアドレスとパスワードでログイン
3. 画面左のメニューから **「管理者機能」** をクリック
4. **「医院アカウント管理」** をクリック
5. 画面右上の **「新規クリニック登録」** ボタンをクリック
6. 用意した1院目の情報を入力
   - クリニック名
   - メールアドレス
   - 住所
   - 電話番号
   - 開業年
7. **「登録」** ボタンをクリック
8. 同じ手順で残り4院も登録

### 確認すること

- [ ] 5つのクリニックが登録できましたか？
- [ ] 一覧画面に5つとも表示されていますか？
- [ ] 登録は簡単でしたか？

**問題があれば**: どこで困ったか教えてください

## ステップ3: 月次データを1件入力する

### やること
1ヶ月分のデータを手入力します。

### 手順

1. 画面左のメニューから **「基礎データ管理」** をクリック
2. **「新規データ登録」** ボタンをクリック
3. 1ヶ月分のデータを入力

   - 売上（例: 5,000,000円）
   - 患者数（例: 150人）
   - スタッフ数（例: 10人）
   - 診療日数（例: 20日）
4. **「保存」** ボタンをクリック
5. 一覧画面に戻って、入力したデータが表示されることを確認

### 確認すること

- [ ] データ入力は簡単でしたか？
- [ ] 1ヶ月分の入力に何分かかりましたか？（目安: 1分以内）
- [ ] 入力したデータは正しく表示されていますか？

**問題があれば**: どの項目が分かりにくかったか教えてください

## ステップ4: CSVファイルで12ヶ月分まとめて登録する

### やること
Excelで12ヶ月分のデータを作って、一気に登録します。

### 準備: Excelファイルを作る

1. Excelまたはスプレッドシートを開く
2. 以下の形式で入力

| 年月 | 売上 | 患者数 | スタッフ数 | 診療日数 |
|------|------|--------|-----------|---------|
| 2025-01 | 5000000 | 150 | 10 | 20 |
| 2025-02 | 4800000 | 145 | 10 | 18 |
| 2025-03 | 5200000 | 155 | 10 | 21 |
| ... | ... | ... | ... | ... |

3. 「CSV UTF-8（コンマ区切り）」で保存
   - ファイル名: `sakura_monthly_data.csv`

### 手順

1. 「基礎データ管理」画面で **「CSV一括登録」** ボタンをクリック
2. 作成したCSVファイルを選択
3. プレビュー画面でデータを確認
4. **「登録」** ボタンをクリック
5. 「登録完了: 12件」と表示されることを確認

### 確認すること

- [ ] CSVファイルは簡単に作れましたか？
- [ ] アップロードはスムーズでしたか？
- [ ] 12ヶ月分のデータが一覧に表示されていますか？

**エラーが出たら**: エラーメッセージを教えてください

## ステップ6: シミュレーションを試す

### やること
「スタッフを2人増やしたら売上はどうなる？」というシミュレーションをします。

### 手順

1. 画面左のメニューから **「経営シミュレーション」** をクリック
2. **「新規シミュレーション」** ボタンをクリック
3. 条件を入力
   - シミュレーション名: スタッフ増員パターン
   - 目標売上: 月商600万円
   - 達成期限: 6ヶ月後
   - 施策: スタッフ2名増員（人件費+50万円/月）
4. **「シミュレーション実行」** ボタンをクリック
5. グラフが表示されることを確認（5秒くらいかかります）

### 確認すること

- [ ] シミュレーションは分かりやすいですか？
- [ ] 結果のグラフは見やすいですか？
- [ ] 予測は納得できる内容ですか？

## ステップ7: レポートを出力する（PDF）

### やること
月次レポートをPDFで出力します。

### 手順

1. 画面左のメニューから **「レポート生成」** をクリック
2. **「月次経営レポート生成」** ボタンをクリック
3. 対象年月を選択（例: 2025年12月）
4. **「生成」** ボタンをクリック
5. 「レポート生成中...」と表示される（10〜20秒待つ）
6. PDFファイルがダウンロードされる
7. PDFを開いて確認

**PDFに含まれる内容**:
- クリニック名、対象年月
- 売上・患者数・診療単価のグラフ
- 前年同月との比較
- 経営指標一覧

### 確認すること

- [ ] PDFは正常にダウンロードされましたか？
- [ ] グラフは綺麗に表示されていますか？
- [ ] 印刷して使えそうですか？
- [ ] 生成時間は20秒以内でしたか？

## ステップ8: スタッフを招待する

### やること
スタッフを招待して、権限を設定します。

### 手順

2. **「スタッフ招待」** ボタンをクリック
3. スタッフ情報を入力
   - メールアドレス（例: staff1@example.com）
   - 権限: **編集者**（データを見る・編集できる）
   - 氏名（例: 山田太郎）
4. **「招待」** ボタンをクリック
5. 「招待メール送信完了」と表示されることを確認

### 確認すること

- [ ] スタッフ招待は簡単でしたか？
- [ ] 権限の意味は分かりやすいですか？

## ステップ9: 印刷物の価格表を登録する

### やること

### 手順

1. 画面左のメニューから **「管理者機能」** → **「価格表管理」** をクリック

   - 基本料金: 10,000円
   - 追加料金: 100円/枚
   - 最小ロット: 100枚
   - 納期: 7営業日
4. **「登録」** ボタンをクリック

- [ ] 価格表登録は簡単でしたか？
- [ ] 一覧に表示されていますか？
