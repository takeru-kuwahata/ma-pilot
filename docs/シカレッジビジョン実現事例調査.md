# シカレッジビジョン実現事例調査レポート

**調査日**: 2026年1月8日
**調査目的**: 「物心両面での幸福、やりがいに満ちた職場、企業貢献、最高の経営支援ツールであり学びの場」を実現している国内事例の徹底調査

---

## 📊 エグゼクティブサマリー

本調査では、医療・介護業界および他業種において、従業員の幸福度向上と経営成果を両立させているプラットフォーム事例を分析しました。以下の共通成功要因が明らかになりました。

### 成功の5つの共通要素

1. **無料または低価格での価値提供** - 最初は無料でコミュニティに参加、学習コンテンツで信頼を構築
2. **実務直結の学習コンテンツ** - すぐに現場で使える5-10分の短尺動画、実践的な知識
3. **双方向コミュニティの構築** - 質問・相談できる場、同業者との交流による孤独感の解消
4. **サプライヤーとの三方良し関係** - 企業が広告・スポンサーとして参加、ユーザーは質の高い情報を無料で取得
5. **データ活用による経営支援** - 学習履歴・業務データを分析し、経営改善につなげる

## 1. 医療・介護業界の学習プラットフォーム

### 1-1. 看護roo!（カンゴルー）- 看護師の「心の拠り所」を創る

**運営**: 株式会社クイック（東証プライム上場）
**会員数**: 120万人以上（看護師・看護学生）
**ビジネスモデル**: 人材紹介事業からの成果報酬 + 広告収益

#### 物心両面の幸福をどう実現しているか

**「心」の幸福**:
- **看護師専用掲示板『ナースカタリーナ』**: 看護師同士が気軽に交流できる場を提供。業務の悩みや疑問を匿名で相談でき、「一人じゃない」という安心感を創出
- **息抜き・共感コンテンツ**: 業務のスキマ時間に読める共感記事、看護あるあるネタで心の余裕を生む
- **キャリア相談**: 転職だけでなく、働き方の悩みにも寄り添うコンテンツ

**「物」の幸福**:
- **無料の学習プラットフォーム『看護roo!まなびチャンネル』**: 2024年4月オープン、9社のパートナー企業と提携
- **23の学習カテゴリー**: 専門知識から最新医療情報まで網羅（2024年7月時点）
- **ユーザー満足度92%**: 実務に直結する質の高いコンテンツ
- **国家試験対策アプリ**: 無料で提供、キャリアアップを支援

#### 学習と経営成果の紐付け

- **転職支援との連携**: 学習で知識を高め→自信をつけ→より良い職場へ転職→定着率向上
- **スキルアップによる昇給**: 専門知識習得により、専門看護師・認定看護師などの資格取得を後押し
- **離職率低下**: 学習機会とコミュニティによる職業満足度向上

#### コミュニティ・エンゲージメント施策

- **ユーザーファースト戦略**: 「目先の売上やPVのためのコンテンツは作らない」という明確な方針
- **長期的な信頼関係**: 「長期的にユーザーに愛されること」をミッション化
- **双方向コミュニケーション**: 掲示板での看護師同士の助け合い文化

#### 企業（サプライヤー）との関係構築

- **パートナーシップモデル**: 医療機器メーカー、製薬会社など9社とパートナー契約
- **Win-Win-Win**:
  - 看護師: 無料で最新の医療知識を習得
  - パートナー企業: 自社製品の認知度向上、医療従事者への直接リーチ
  - 看護roo!: プラットフォーム価値向上、収益の多角化

#### 成功要因と数値実績

- **会員数120万人超**: 看護師免許保有者の約1/10が会員
- **継続的な過去最高益更新**: 運営会社クイックは東証プライム上場企業として安定成長
- **ブランド力**: 「看護師といえば看護roo!」という認知度
- **徹底した本人確認**: 医療従事者限定により、質の高い情報交換を担保

### 1-2. メドピア（MedPeer）- 医師の3人に1人が参加するプラットフォーム

**運営**: メドピア株式会社（東証グロース上場）

- **専門外の知識習得**: 他科の医師との交流で視野が広がる

**「物」の幸福**:
- **年間12万円の学習支援**: 書籍・研修・カンファレンス等の学習を幅広く支援（職種を問わず）

- **経営支援SaaS**: 薬局・病院・クリニック向けの「やくばと」「kakari」シリーズで、医療機関の経営を支援

#### 学習と経営成果の紐付け

- **診療報酬改定対応**: 経営支援SaaSで、制度変更にも迅速に対応

#### コミュニティ・エンゲージメント施策

- **医師同士のピアレビュー**: お互いに知見を評価し合う文化

#### 企業（サプライヤー）との関係構築

- **製薬企業向けマーケティング支援**: MedPeerプラットフォームを活用し、医師に直接リーチ
- **医療機器企業向けコンテンツ制作**: 専門家監修の質の高いコンテンツ提供
- **BPO事業**: コントラクトMR/MSLサービスで、製薬企業の営業活動を支援

#### 成功要因と数値実績

- **国内医師の3人に1人**: 圧倒的なリーチ
- **営業利益率8.6%**: 計画を大きく超過（2024年度）
- **選択と集中**: 成長への貢献が高い事業への集中投資を継続
- **構造改革の推進**: 事業撤退等により収益率改善

## 6. シカレッジへの示唆

### 6-1. 即座に取り入れるべき要素

#### 1. 無料学習コンテンツの充実
- **参考事例**: 看護roo!まなびチャンネル、ホットペッパービューティーアカデミー
- **施策**: まずは無料で価値提供、信頼構築後に有料サービスへ誘導

#### 2. 質問・相談できるコミュニティ
- **参考事例**: 美容師オンラインサロンの質問箱、看護roo!のナースカタリーナ
- **施策**: 匿名で気軽に質問できる仕組み、歯科医師・歯科衛生士同士の助け合い文化

#### 3. 短尺動画での実務直結コンテンツ
- **参考事例**: ジョブメドレーアカデミー（5分動画）、Schoo（月50本更新）
- **施策**: 5-10分の動画、スキマ時間で学習可能、スマホ対応

#### 4. 学習と経営成果の見える化
- **参考事例**: カイポケの経営管理機能、サイボウズのチームワーク総研
- **施策**: 学習履歴と経営数値（売上、利益率、患者満足度）の相関を可視化

#### 5. サプライヤーとの三方良し
- **参考事例**: 看護roo!のパートナーシップ、メドピアの製薬企業連携
- **施策**: 歯科材料メーカー、歯科機器メーカーをスポンサーに、歯科医院は無料で最新情報取得

- **反面教師**: 企業宣伝ばかりで学習価値が低い

- **反面教師**: 匿名の誹謗中傷、炎上
