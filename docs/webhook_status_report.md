## WordPressアカウント自動発行について

WordPressへの接続情報がサーバー（Render.com）に未登録のため、現在はWordPressへのアカウント発行がスキップされています。
以下の3つの情報をRender.comのサーバー設定に追加していただくと、自動発行が可能になります。

| 項目 | 内容 |
|------|------|
| WordPressのサイトURL | 例：https://shika-college.com |
| WordPress管理者ユーザー名 | WordPress管理画面のログインユーザー名 |
| WordPressアプリケーションパスワード | WordPress管理画面 → ユーザー → プロフィール → 「アプリケーションパスワード」で新規発行 |

この設定はこちら（開発側）で行いますので、上記3点の情報をご共有いただければ対応いたします。

## Lステップ側の設定（御社にてご対応をお願いします）

Lステップの管理画面で、下記のWebhook URLを登録してください。
登録先のフォームは、内覧会申込フォーム（フォームID: 710696）です。

**Webhook URL（登録するURL）:**

```
https://ma-pilot.onrender.com/api/webhooks/lstep
```

登録手順については、Lステップのサポートドキュメントまたはサポートチャットをご確認いただくか、ご不明な場合はお申し付けください。
