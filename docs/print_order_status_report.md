# 印刷物発注機能 実装状況ご報告

## 今回対応したこと

### 1. 注文通知メールに商品・数量を表示

これまで注文通知メール（クリニック宛の受付確認メール）に商品種類・数量が記載されない場合がありました。相談フォームからの注文も含め、すべてのパターンで商品情報がメール本文に表示されるよう修正しました。

### 2. 通知メールの件名を修正

相談フォーム・再注文・新規注文のいずれの場合も件名が「新規注文」と表示されていた問題を修正しました。

修正後の件名：
- 相談フォームからの注文：【相談フォーム】〇〇医院 様から印刷物のご注文
- 再注文の場合：【再注文】〇〇医院 様から印刷物のご注文
- 新規注文の場合：【新規注文】〇〇医院 様から印刷物のご注文

### 3. 発注履歴下部に価格改定の注意書きを追加

印刷物発注履歴ページの下部に以下の注意書きを追加しました。

「※物価の高騰や原材料費の変動により、予告なく価格が変更となる場合がございます。あらかじめご了承ください。」

## 商品価格・内容の管理について

商品の価格・納期・表示順の編集は、MA-Pilot管理者画面から行うことができます。

**操作手順：**

1. MA-Pilotにシステム管理者アカウントでログイン
2. 右上の「運営者モードへ」ボタンをクリック
3. 左メニューの「価格表管理」をクリック
4. 各商品の価格・デザイン費・納期を編集、または新規追加・削除が可能です

変更内容は即座に発注フォームに反映されます。価格・商品内容は全医院共通です。
