## ❓ よくある質問

### Q1. 診療圏分析なしでも使えますか？
**A.** はい、問題なく使えます。経営ダッシュボード・データ管理・シミュレーション・レポート生成など主要機能は全て動作します。診療圏分析は追加機能として2月に実装します。

### Q2. 外部APIキーの取得は必須ですか？
**A.** 診療圏分析を使わない場合は不要です。Phase 2で追加実装する際に取得いただければ問題ありません。

### Q3. Google Maps API Keyでお金がかかることはありますか？
**A.** 月10,000リクエストまで無料です。通常の使用範囲（月100〜200クリニック分析）では無料枠内に収まります。

### Q4. 今日の確認で重大な問題が見つかったら？
**A.** 問題の重大度によっては正式リリースを延期します。ただし、軽微な修正であれば1/30中に対応可能です。

### Q5. Phase 2をスキップして、Phase 6を先に実装できますか？
**A.** 可能です。クライアントのご要望に応じて実装順序を調整できます。

### Q6. テストデータは何院分必要ですか？
**A.** 最低1院あれば動作確認可能です。ただし、複数院のデータ分離やグラフの見栄えを確認するため、5院分を推奨します。
