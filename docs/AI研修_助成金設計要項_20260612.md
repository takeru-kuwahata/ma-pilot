# AI研修 × 人材開発支援助成金 設計要項

作成日：2026年6月12日
作成：日本医療デザインセンター 桑畑
宛先：メディカルアドバンス 本多社長

---

## 使用する制度

**人材開発支援助成金（事業展開等リスキリング支援コース）**

| 項目 | 内容 |
|---|---|
| 所管 | 厚生労働省 |
| 申請主体 | メディカルアドバンス（雇用保険適用事業所） |
| 経費助成率 | 75%（中小企業） |
| 賃金助成 | 1,000円/時間/人 |
| 1人あたり上限 | 30万円（集合研修・50時間未満） |
| 訓練形式 | 集合研修（OFF-JT）として申請 |

---

## 研修形式

| 項目 | 内容 |
|---|---|
| 提供会社 | 日本医療デザインセンター（外部訓練機関） |
| 形式 | ハイブリッド型（東京現地 + 同時双方向リモート） |
| 根拠 | 同時双方向型通信訓練は集合研修と同等（厚労省公式） |
| 頻度 | 集合研修：月2コマ（3h×2）＋ グループコンサル：月3回（1h×3） |
| 期間 | 6ヶ月 |
| 対応人数 | 最大20名（品質維持可能） |
| サポート | MDCアシスタントスタッフによるテクニカルサポートあり |

---

## 研修の目的・ゴール

メディカルアドバンスの選抜メンバーが、AIを活用したDX推進を自走できる人材となり、自社サービスを開発して歯科医院に提供できるようになること。

## 契約フロー

```
メディカルアドバンス
  → 日本医療デザインセンターへ月額支払い（受講費用として請求）
      └── MDC手取り：32万円
      └── Claude Team Premium原価：人数×1.5万円
  → 助成金申請（社労士が代行）
      └── 受給後に社労士成功報酬を支払い
```

## スコープ

### 含むもの
- AI・DX推進の研修（6ヶ月プログラム）
- 週1回グループコンサルティング（オンライン）
- Claude Team Premiumの環境提供（Claude Code含む）
- MDCアシスタントによるテクニカルサポート

### 含まないもの
- Webサイト改修・システム開発の受託
- 開発案件は別途見積もり、または自社開発研修の成果として対応

## 注意事項

**社労士について**
社労士への支払いは弊社側で契約している社労士前提での金額になります。助成金に強みがあり、実績もある社労士のため相場よりも割高になっています。社労士を御社で選定することも可能ですのでお申し出ください。

**年度内完了について**
助成金の申請・受給には年度内の完了が必須となります。研修開始日から逆算して訓練計画届の提出（開始1ヶ月前まで）が必要です。開始時期はお早めにご相談ください。

**プラン・日程のカスタマイズについて**
標準は6ヶ月プログラムですが、最短3ヶ月・最長8ヶ月のプランが作成可能ですのでお申し出ください。また、1ヶ月の中での実施日数・実施時間もご相談ください。
